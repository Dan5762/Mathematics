\section*{Matrices and Vector Spaces}

\subsection*{Conditions for a Vector Space}

\begin{enumerate}[label=(\roman*)]
\item Vector set is closed under commutative and associative addition
\item Vector set is closed under distributative and associative multiplication by a scalar
\item Vector set contains a null vector
\item Multiplication by unity leaves any vector unchanged
\item All vectors have a corresponding negative 
\end{enumerate}

\textline{Span of a vector space:}{The span of a vector space is defined as the set of all vectors that can be expressed as a linear sum of the basis vectors:
